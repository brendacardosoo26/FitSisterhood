Política de Privacidade - FitSisterhood

1. Introdução

Bem-vinda ao FitSisterhood! Sua privacidade é muito importante para nós. Esta Política de Privacidade explica como coletamos, usamos, armazenamos e protegemos suas informações pessoais quando você usa nosso aplicativo.

Ao usar o FitSisterhood, você concorda com a coleta e uso de informações de acordo com esta política.

2. Informações que Coletamos

2.1 Informações Fornecidas por Você

Coletamos as seguintes informações quando você usa o aplicativo:

  • Dados de Ciclo Menstrual: Data de início da menstruação, fase do ciclo, sintomas relatados
  • Dados de Treino: Tipo de atividade física, duração, intensidade, calorias estimadas
  • Dados Nutricionais: Refeições registradas, calorias, macronutrientes
  • Dados de Progresso: Peso, medidas corporais, fotos de progresso (se você optar por adicionar)
  • Preferências: Objetivos fitness (emagrecimento, ganho de massa, ambos)

2.2 Informações Coletadas Automaticamente

  • Dados de Uso: Telas visitadas, funcionalidades utilizadas, tempo de uso do app
  • Informações do Dispositivo: Modelo do dispositivo, versão do sistema operacional, identificador único do dispositivo
  • Dados de Compra: Histórico de compras de receitas premium, Desafio 21 e assinaturas (gerenciado pelas lojas de app)

2.3 Informações que NÃO Coletamos

  • ❌Não coletamos dados de localização GPS
  • ❌Não coletamos contatos ou lista de amigos
  • ❌Não coletamos informações de outros aplicativos
  • ❌Não vendemos seus dados para terceiros

3. Como Usamos Suas Informações

Utilizamos suas informações para:

  • Fornecer funcionalidades do app: Rastrear seu ciclo menstrual, treinos e nutrição
  • Personalizar sua experiência: Oferecer dicas baseadas na fase do seu ciclo
  • Melhorar o aplicativo: Analisar padrões de uso para aprimorar funcionalidades
  • Processar compras: Gerenciar suas compras de receitas premium, Desafio 21 e assinaturas (via In-App Purchase)
  • Enviar notificações: Lembretes de treino, hidratação e dicas motivacionais (se você permitir)
  • Suporte ao cliente: Responder suas dúvidas e resolver problemas

4. Como Armazenamos Suas Informações

4.1 Armazenamento Local

A maioria dos seus dados é armazenada localmente no seu dispositivo usando AsyncStorage:

  • Dados de ciclo menstrual
  • Registros de treinos
  • Registros nutricionais
  • Fotos de progresso
  • Preferências do aplicativo
Importante: Se você desinstalar o aplicativo, esses dados serão permanentemente excluídos do seu dispositivo.

4.2 Armazenamento em Nuvem (Opcional)

Se você optar por criar uma conta e sincronizar seus dados:

  • Dados são armazenados em servidores seguros com criptografia
  • Você pode acessar seus dados de múltiplos dispositivos
  • Backups automáticos protegem contra perda de dados

4.3 Segurança

Implementamos medidas de segurança técnicas e organizacionais para proteger seus dados:

  • Criptografia de dados em trânsito (HTTPS/TLS)
  • Criptografia de dados em repouso
  • Acesso restrito a dados pessoais
  • Monitoramento de segurança contínuo

5. Compartilhamento de Informações

5.1 NÃO Compartilhamos Seus Dados Com:

  • ❌ Empresas de publicidade
  • ❌ Corretores de dados
  • ❌ Redes sociais (exceto se você escolher compartilhar)
  • ❌ Seguradoras de saúde

5.2 Compartilhamento Limitado:

Podemos compartilhar informações apenas nas seguintes situações:

  • Lojas de App (Apple App Store e Google Play): Para processar compras via In-App Purchase (as lojas gerenciam pagamentos)
